# Contributing

Thank you for your interest in POCO Toolkit.

This project is currently maintained by its creator and is under active development.

## Project Philosophy

Every device is analyzed before modifications are recommended.

Workflow:

Scan → Analyze → Debloat → Restore → Verify

## Reporting Issues

When reporting an issue, please include:

- Device model
- HyperOS version
- Android version
- Scan report (Phone_Scan.txt)
- Description of the problem

## Suggestions

Feature requests and testing feedback are always welcome.

Please avoid sharing generic debloat lists without a device scan report.

Safety and reversibility are the project's highest priorities.
